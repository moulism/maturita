Transformace dat pomocí knižovny Pandas

⏱ ~15 minut
🎯 Klíčové pojmyDataFrame, read_csv, head/info/describe, filtrace, groupby, merge, pivot_table, čistění dat

1. Import a první průzkum dat

import pandas as pd

df = pd.read_csv('data.csv', sep=';', encoding='utf-8-sig')
df.head()          # prvních 5 řádků
df.tail()          # posledních 5
df.info()          # typy sloupců + NULL hodnoty
df.describe()      # statistiky číselných sloupců
df.shape           # (počet_řádků, počet_sloupců)
df.columns         # seznam sloupců
df.isnull().sum()  # počet NULL v každém sloupci

2. Filtrace

# Horizontální (výběr řádků):
df[df['vek'] > 18]
df[(df['mesto'] == 'Praha') & (df['vek'] > 25)]
df.query('vek > 18 and mesto == "Praha"')

# Vertikální (výběr sloupců):
df[['jmeno', 'vek', 'mesto']]

3. Transformace

df['cena_s_dph'] = df['cena'] * 1.21
df['mesto'] = df['mesto'].str.strip().str.upper()
df['datum'] = pd.to_datetime(df['datum'])

4. Agregace – GroupBy

df.groupby('kategorie')['cena'].sum()
df.groupby('kategorie').agg({'cena': 'mean', 'id': 'count'})

5. Merge (JOIN) a Pivot

pd.merge(df_zak, df_obj, on='id', how='left')

df.pivot_table(values='trzby', index='region',
               columns='mesic', aggfunc='sum')

6. Čistění dat

df.dropna()                              # smaž řádky s NULL
df['vek'].fillna(df['vek'].mean())       # nahrad NULL průměrem
df.drop_duplicates(subset=['id'])       # odstraň duplikace
df['cena'] = df['cena'].astype(float)    # změň typ
💡 Tip na 15 minutVždy pracuj s kopií: df = df.copy(). Metody lze řetězit: df.query(...).rename(...).groupby(...).agg(...). Při exportu do CSV: index=False, encoding='utf-8-sig' – jinak se v Excelu rozsype čeština.
