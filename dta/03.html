Spuštění dockerizované databáze a extrakce dat

⏱ ~15 minut
📝 Potítko – napiš si na papír
  • Docker = kontejner (izolované prostředí) spustitelný na jakémkoliv OS
  • docker-compose up -d spustí DB na pozadí
  • Connection string: postgresql://user:password@localhost:5432/dbname
  • pip install psycopg2-binary – driver pro PostgreSQL (bez něj SQLAlchemy neumí mluvit s Postgres)
  • pd.read_sql_query(sql, engine) – SQL dotaz přímo do DataFrame

Co je Docker a proč ho používáme?

Docker zabalí aplikaci i se všemi závislostmi do kontejneru. Ten běží stejně na Windows, Linuxu i Macu – žádné "u mě to funguje". Pro analytiku: PostgreSQL databáze poběží na tvém notebooku bez nutnosti instalace.

Image
Šablona (recept). Např. postgres:15 – hotová image PostgreSQL. Stahuje se z Docker Hub.
Kontejner
Spuštěná instance image. Z jedné image lze spustit N kontejnerů.
docker-compose.yml
Konfigurační soubor – popisuje, jaké kontejnery spustit, na jakých portech, s jakými env proměnnými.

docker-compose.yml + spuštění

# docker-compose.yml
version: '3.8'
services:
  postgres:
    image: postgres:15
    environment:
      POSTGRES_USER: user
      POSTGRES_PASSWORD: password
      POSTGRES_DB: dbname
    ports:
      - "5432:5432"
docker-compose up -d     # spustí na pozadí
docker-compose ps        # stav kontejnerů
docker-compose down      # zastavení

Připojení Pythonu k databázi

from sqlalchemy import create_engine
import pandas as pd

engine = create_engine('postgresql://user:password@localhost:5432/dbname')
df = pd.read_sql_query("SELECT * FROM objednavky WHERE rok = 2024", engine)
df.head()

⚠ Chytáky a záludné otázky
  • Co je rozdíl Docker image vs kontejner? – Image je jako ISO soubor (statická šablona). Kontejner je spuštěná instance – jako spuštěné PC z ISO. Z jedné image lze spustit 100 kontejnerů.
  • Proč SQLAlchemy a ne přímo psycopg2? – SQLAlchemy je abstrakční vrstva – stejný kód funguje s PostgreSQL, MySQL i SQLite, stačí změnit connection string.
  • Jaký je formát connection stringu? – Vždy: dialect://user:password@host:port/dbname. To je nutné umět zpaměti.
💡 Tip na 15 minut — říkej komisi Klíč celé otázky je connection string. Formát: dialect://user:password@host:port/dbname. Nejčastější chyba: zapomenout nainstalovat driver – pip install psycopg2-binary. SQLAlchemy samo neumí mluvit s PostgreSQL, potřebuje driver. Druhá záludnost: při exportu do CSV vždy encoding=utf-8-sig, jinak se čeština v Excelu zobrazí jako nesmysl.
