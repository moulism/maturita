Typy dat a složené datové typy v Pythonu

⏱ ~15 minut
📝 Potítko – napiš si na papír
  • Kvalitativní: Nominální (bez pořadí: barva) / Ordinální (s pořadím: známky)
  • Kvantitativní: Diskrétní (celá čísla: počet kusů) / Spojitá (desetinná: cena, váha)
  • List [měnitelný, seřazený], Tuple (neměnný), Set {unikátní}, Dict {klíč: hodnota}
  • Atomické: int, float, str, bool, NoneType (= SQL NULL)
  • Deterministický = stejný vstup → vždy stejný výstup

⚠ Chytáky a záludné otázky
  • Rozdíl List vs Tuple? – List je měnitelný (append, remove fungují). Tuple je neměnný – po vytvoření ho nelze změnit. Tuple je rychlejší a signalizuje úmysl: tato data se nemění.
  • Kdy nominální a kdy ordinální? – Nominální = bez pořadí (barva očí – modrá není více než hnědá). Ordinální = s pořadím, ale bez přesných mezer (1 je lepší než 2, ale nevíme o kolik).
  • Proč None a ne 0 nebo prázdný string? – None signalizuje absenci hodnoty – 'nevím, co tam je'. 0 nebo '' jsou konkrétní hodnoty. Pro analytiku je crucial rozdíl mezi '0 objednávek' a 'nevíme kolik objednávek má'.
💡 Tip na 15 minut — říkej komisi Datové typy rozdělím na statistické (kvalitativní: nominální/ordinální; kvantitativní: diskrétní/spojitá) a programátorské. V Pythonu jsou čtyři základní kolekce: List pro uchování pořadí, Tuple pro neměnná data, Set pro unikátní hodnoty, Dict pro vyhledávání klíč-hodnota. Klíčový detail: Set automaticky odstraní duplicity – len(set(df['produkt'])) vrátí počet různých produktů.
