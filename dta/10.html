Funkce a moduly v Pythonu

⏱ ~15 minut
📝 Potítko – napiš si na papír
  • def nazev(a, b): – definice; return vrátí hodnotu
  • Funkce (má return) vs Procedura (nemá return, jen provede akci)
  • Type hints: def scitani(a: int, b: int) -> int:
  • Lambda: lambda x: x * 2 – anonymní jednořádková funkce
  • Modul = soubor .py s funkcemi; import modul nebo from modul import fce

Funkce – proč je používáme

Funkce zabraňuje opakování kódu (DRY – Don't Repeat Yourself). Pokud budeš počítat DPH na 10 místech skriptu, stačí opravit jednu funkci. Navíc kód je čitelnější: add_vat(df) říká víc než 3 řádky výpočtu.

Základní syntaxe

def scitani(a: int, b: int) -> int:
    return a + b

vysledek = scitani(3, 4)   # 7

def tiskni_pozdrav(jmeno: str) -> None:  # procedura
    print(f"Ahoj, {jmeno}!")

def vypocti_dph(cena: float, sazba: float = 0.21) -> float:  # výchozí hodnota
    return cena * (1 + sazba)

Try/except uvnitř funkce

def bezpecne_deleni(x: float, y: float) -> float | None:
    try:
        return x / y
    except ZeroDivisionError:
        print("Nelze dělit nulou!")
        return None

Lambda – anonymní funkce

double = lambda x: x * 2
double(5)   # 10

# Nejčastější použití jako argument jiné funkce:
cisla.sort(key=lambda x: -x)             # sestupné řazení
df['cena_dph'] = df['cena'].apply(lambda x: x * 1.21)

Moduly – organizace kódu

# transformace.py:
def add_vat(df, col):
    df[col + '_s_dph'] = df[col] * 1.21
    return df

# Importování:
import transformace
transformace.add_vat(df, 'cena')

from transformace import add_vat
add_vat(df, 'cena')

from transformace import add_vat as dph  # alias

Pravidla pojmenování

Povoleno
Písmena, číslice (ne na začátku), podtržítko. vypocti_dph, _private
Zakázáno
Začátek číslem, speciální znaky, diakritika, Python klíčová slova (def, class)
⚠ Chytáky a záludné otázky
  • Rozdíl funkce vs procedura? – Funkce vrací hodnotu pomocí return. Procedura jen provede akci, nic nevrací (vrátí None). V Pythonu není syntaktický rozdíl – záleží jen na tom, zda má return.
  • Co jsou type hints? – Anotace parametrů a návratové hodnoty. Python je nevynucuje, ale pomáhají IDE a linterům odhalit type-chyby a zpřehledňují kód.
  • Kdy nepoužít lambdu? – Lambda je vhodná pro jednoduché jednořádkové výrazy. Pokud potřebuješ více řádků, podmínky nebo try/except – použij normální def.
💡 Tip na 15 minut — říkej komisi Funkce je základní stavební blok – DRY princip. Rozlišuji funkci (má return, vrací hodnotu) od procedury (jen provede akci). Type hints Python nevynucuje, ale jsou dobrá praxe. Lambda použiji pro jednoduché výrazy v apply() nebo sort() – pro složitější logiku vždy def.
